Scope of work
Planning and Requirements Gathering:
o Complete requirements gathering and analysis of Hamad Bin Khalifa University requirements.
o Prepare Project Plan, Risk Mitigation Plan, Change Management Plan, Communication Plan and Stakeholder’s Availability.

Licenses Installation and Implementation:
o Check readiness of the Servers and Operating system licenses.
o Deploy and configure eversuite solutions on the production, and staging/ testing environments

Configuration of eversuite.document
o Documents Categories Configuration
o Security Context Definition (Groups, Roles, Security Rights)
o Configure Document Collaboration
o Configure Audit Timeline
o Configure Document Timeline
o Configure Indexing Engine
o Configure Reports
o Configure Alerts and Reminders
o Configure Document Search
o Configure Document Review Functionality
o Configure Office Online
o Enable E-mail Notifications

Integration:
Ever Gulf Co. provides a set of readymade connectors built with Restful APIs in order to connect with:
o Microsoft Active Directory
o Microsoft Office
o Banner Student Information System
o Canvas Learning Management System
o Sierra Library System
o QF ERP
o Oauth and SAML
o SharePoint

Solution Language:
All Solution modules to be available in both languages’ Arabic and English

Deployment:
Deploy the solution on Staging\Testing environment.

Internal Testing:
o Unit Testing
o Performance Testing
o Functional and Integration Testing
o System Testing
o Security Penetration Testing
